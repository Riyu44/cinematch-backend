Machine Learning
3-model unsupervised pipeline
No labelled training data. All three models discover structure from the data itself — content similarity, genre clusters, and latent taste patterns.

KNN on Sentence Embeddings

Every movie's title, genres, and overview are encoded into a 384-dimensional vector using all-MiniLM-L6-v2. User's watched movies are averaged into a taste vector and the 20 nearest neighbours are retrieved by cosine similarity.

Weight in ensemble: 50%

K-Means Clustering

All 1,389 movies are grouped into 20 mood/genre clusters using genre one-hot encoding + normalised rating, runtime, and popularity. User preferences pre-filter the candidate pool from 1,389 → ~70 before scoring.

Weight in ensemble: 20% + pre-filter

Collaborative Filter (TruncatedSVD)

A user-movie affinity matrix is factorised into 50 latent taste dimensions. New users are projected into this space via their watched movies to find taste-similar users and borrow their top-rated unseen titles.

Weight in ensemble: 30%
